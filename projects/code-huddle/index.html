Code Huddle

Fullstack Developer

2017 - A single page application for conducting coding interviews. This application features a collaborative coding and whiteboard environment. Built as a capstone project for the Fullstack Academy Software Engineering Immersive program. Developed with Node.js, React, Redux, Express, Sequelize, Mocha, Chai, and Socket.io. As a member of the team, I designed the data flow for Redux between the front and back-end with websockets.
